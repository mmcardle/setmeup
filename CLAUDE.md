# setmeup

A tool for setting up a new development machine.

## Features

- Install system packages
- Install development tools
- Install dotfiles
- Install shell configurations
- Install git configurations
- Install ssh configurations
- Install mise configurations

## Tests

To run the tests, run the following command:

```sh
./tests/run_tests.sh
```

## Development

Always update the tests when making changes to the code.

Run the tests before completing the users requests.
